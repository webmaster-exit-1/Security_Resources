# Security Resources

A curated collection of security tools, learning materials, and references for security professionals, researchers, and enthusiasts.

## 🌐 GitHub Pages

The project is published as a GitHub Pages website. Visit the live page at:

**https://webmaster-exit-1.github.io/Security_Resources/**

## Contents

The site covers:

- **Security Tools** – Nmap, Metasploit, Wireshark, Burp Suite, SQLMap, Kali Linux, and more
- **Learning Resources** – OWASP, TryHackMe, Hack The Box, PortSwigger Web Security Academy, SANS, CTFtime
- **Quick Reference** – NVD, CVE/MITRE, MITRE ATT&CK, NIST SP 800-53, CISA, Exploit-DB
